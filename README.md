# Juego de Ahorcado en JAVA



>Hicimos un  juego de ahorcado en java donde uno debe ir colocando letra a letra en la consola y tiene cierta cantidad de intentos. si lo logra completar toda la palabra uno gana, en caso contrario, se acaben los intentos uno pierde.
>
## ¿Qué herramientas utilizamos?
1.	Clase Scanner
2.	Declaraciones y asignaciones
3.	Arreglos
4.	Estructura de control: Iterativa(Bucle)
5.	Clase String y sus métodos valueOf, charAt y length()
6.	Clase Character y su método toLowerCase
7.	Estructura de control condicional
